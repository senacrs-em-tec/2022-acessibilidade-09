Tabindex

Tendo em vista que as pessoas com deficiência visual utilizam a tecla tab para navegar em sites, nos utilizamos o atributo do html tabindex. Caso um componente tenha seu tabindex igual a 0, ele pode ser selecionado pela tecla.

<footer>
<a href="#" tabindex="0">Inicio</a>
<p tabindex="0">Arthur Costa, Arthur da Silva, Davi Cunda e Fernando Campanaro</p>
</footer>

Números acima de 0 são usados para definir a ordem em que os componentes serão selecionados, do contrário o navegador irá definir automaticamente (de cima para baixo e da esquerda para a direita).

Title e alt text

Pessoas que possuem deficiência visual utilizam leitores de tela, porem esses leitores não são capazes de descrever uma imagem para o usuário por isso utilizamos o atributo Alt text. Esse atributo é utilizado para definir o texto que será lido pelo leitor de tela.

Para utilizar o alt text basta escrever ‘alt= “ “’ depois do link da imagem e digitar dentro das aspas a descrição desejada.

alt="Imagem de uma mulher cega com seu cachorro guia se divertindo no parque."

Como alguns navegadores não reconhecem esse atributo, é necessário utilizar também o atributo tilte que, nesse caso, funciona da mesma maneira.

title="Imagem de uma mulher cega com seu cachorro guia se divertindo no parque."

Aria-label

Muitas vezes, queremos que o leitor de tela leia algo diferentes do que está visível para usuários que não o utilizam. Nesse caso, utilizamos o atributo aria-label. Esse atributo funciona como um alt-text, mas que pode ser utilizado em qualquer elemento da página, enquanto o anterior só irá funcionar em imagens. Para utilizar esse elemento, basta escrever ‘aria-label=” ”’ e digitar o texto desejado dentro das aspas.
